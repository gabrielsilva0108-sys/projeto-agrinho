# Agrinho 2026 - Conexão Campo e Cidade

Este projeto foi desenvolvido para o programa Agrinho 2026 com o tema **Conexão campo e cidade**.

## 🌎 Sobre o projeto
O site mostra a importância da relação entre o campo e a cidade, destacando como eles se complementam na produção de alimentos, tecnologia, economia e qualidade de vida.

## 💡 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 🚀 Objetivo
Mostrar de forma simples e interativa como o campo e a cidade estão conectados no dia a dia das pessoas.
